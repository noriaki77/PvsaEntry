アドオンを使う場合は設定１を
Inn:''
のように空にします。

このアドオンのサブスクリプトは画像や固定文字を表示する場所に、 文字や画像をBlink表示します。

IEがBlinkをサポートしてないのでスクリプトによるBlinkです。

<link rel="stylesheet" type="text/css" href="./sample5.css">
<script type="text/javascript" src="./sample5.js"></script>
<script type="text/javascript" src="./sample5ad.js"></script>

■複数ページで共通ファイルを使いながらアドオンスクリプトの部分設定を変更するには、 アドオンスクリプトの最後の
Cset.addon.set(Cset.addon);を外して。
<link rel="stylesheet" type="text/css" href="./sample5.css">
<script type="text/javascript" src="./sample5.js"></script>
<script type="text/javascript" src="./sample5ad.js"></script>
<script type="text/javascript">
Cset.addon.delay = 300;
Cset.addon.set(Cset.addon);
</script>
の様に、Cset.addon の下のプロパティ名(設定項目名)を上書きする形で書きこみ、 最後にCset.addon.set(Cset.addon);を書きこみます。
